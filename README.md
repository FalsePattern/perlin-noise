# Perlin Noise

## About

A tool to generate Perlin Noise (1, 2 and 3D).

## Example

```JavaScript
import PERLIN from './perlin';

// E.g.
let x = 3;
let y = 6;

let noise_2d = PERLIN.noise2D(x, y);
```

## Features

`PERLIN` functions:

- 1D `noise1D(x)`
- 2D `noise2D(x, y)`
- 3D `noise3D(x, y, z)`
